# Project Handoff

## Project

`game-rewind-uk` is a simple static website for UK retro gamers. The idea is a "time capsule" experience that helps visitors see what UK culture was like when a specific game released.

## Tech Setup

- Static site
- Main files:
  - `index.html`
  - `app.js`
  - `about.html`
  - `styles.css`
  - `home.css`
  - `game-rewind-data.js`
- Assets are stored directly in the repo
- No build step was identified in this project
